# Mutual Fund Analytics

## Overview
The Mutual Fund Analytics project is designed to facilitate the analysis and visualization of mutual fund data. This project structure is organized to streamline the workflow from raw data collection to processed data analysis and reporting.

## Project Structure
- **data/raw**: Contains raw data files in formats such as CSV or JSON that have not been processed.
- **data/processed**: Stores cleaned and transformed data files ready for analysis.
- **notebooks**: Contains Jupyter notebooks for data analysis and visualization.
- **sql**: Holds SQL scripts for database queries, schema definitions, and data manipulation.
- **dashboard**: Includes files related to the dashboard for visual data representation.
- **reports**: Stores generated reports summarizing the findings from the data analysis.
- **.gitignore**: Specifies files and directories to be ignored by Git, such as temporary files and logs.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd mutual-fund-analytics
   ```
3. Install necessary dependencies (if applicable).

## Usage Guidelines
- Place raw data files in the `data/raw` directory.
- After processing the data, save the cleaned files in the `data/processed` directory.
- Use the notebooks in the `notebooks` directory for analysis and visualization.
- SQL scripts can be found in the `sql` directory for data manipulation.
- The dashboard files are located in the `dashboard` directory for creating visual representations.
- Generated reports should be saved in the `reports` directory.
